Initialize Mongoloid:

In a Rails initializer:

<pre>
  Mongoloid.connect_to("myapp_database_name")
</pre>

Specifying a Model as a Document in MongoDB:

<pre>
  class Person < Mongoloid::Document
    collection_name :people
    fields \
      :first_name,
      :middle_name,
      :last_name
  end
</pre>
